If you are a user:
Click 'Problems', all problems will be listed by problem id.
Choose one, and problem discription will be shown.
Click 'Submit', and then select the compiler you wanted, copy your code into text area, then submit your source code.
The page will show you the result: compilation error, wrong answer or accepted.



If you are an administrator:
Click 'Admin tools', a page for adding problem will be shown.
The system will assign a problem id for the new problem and it cannot be changed.
Copy the problem discription in, enter the authentication code(password) in, then submit.
Find the directory in htdocs/oj/library/{problem id}/, copy your test data here, name as '{problem id}_n.in' and '{problem id}_n.out'.
Test data must be in pairs.
